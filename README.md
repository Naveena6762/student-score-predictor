# 🎯 Student Score Predictor - Streamlit App

This is a simple Machine Learning-powered web app built with **Streamlit**.  
It predicts a student's final exam score based on:

- 📚 Hours Studied  
- 📅 Attendance Percentage  
- 📊 Previous Exam Score
